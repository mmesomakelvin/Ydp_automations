# 3. Reminders, Nudges, and Weekly KPI Digest Tech Stack

## Purpose

Automate monthly program reminders, no-show nudges, feedback form follow-up, and stakeholder KPI reports.

## Version 1 Stack

| Layer | Tool | Why |
| --- | --- | --- |
| Operational data | Google Sheets | Keeps the system easy for the YDP team to inspect and update. |
| Automation runtime | Google Apps Script V8 | Best fit for scheduled jobs that read Sheets and send emails. |
| Scheduled jobs | Apps Script time-driven triggers | Runs reminder checks, no-show checks, feedback checks, and report generation on a schedule. |
| Email sending | `MailApp.sendEmail` | Sends reminders, nudges, and stakeholder digest emails. |
| Active mentee source | `Matched Pairs` sheet | Tracks confirmed active mentees, not every applicant. |
| Event source | `Program Calendar` sheet | Stores monthly check-ins, guest speaker sessions, and feedback deadlines. |
| Attendance source | Reusable Google Attendance Form response sheet | Used to detect who attended live monthly sessions. |
| Feedback source | Reusable Monthly Feedback Form response sheet | Used to track learning updates, projects, blockers, and mentor ratings. |
| Dashboard | Google Sheets dashboard | Easier to automate and export as PDF than Looker Studio in version 1. |
| Charts | Google Sheets charts | Can be used for dashboard snapshots and PDF reports. |
| PDF report | Google Sheets export workflow | Generates a PDF only when a stakeholder report is sent. |
| Narrative summary | Gemini API | Writes executive summary, risks, observations, and recommended actions from already-calculated metrics. |
| Secrets/config | Apps Script `PropertiesService` | Stores Gemini API key and report settings without hardcoding them in the script. |
| Deployment | A separate Apps Script project via `clasp.cmd` | Keeps operational reminders separate from registration automations. |
| Version control | GitHub | Tracks scripts, artifacts, and architecture decisions. |

## Planned Sheet Tabs

| Sheet/Tab | Purpose |
| --- | --- |
| `Matched Pairs` | Source of active mentees and mentor assignments. |
| `Program Calendar` | Source of upcoming workshops, guest speaker sessions, and feedback deadlines. |
| `Attendance Responses` | Source of attendance form submissions. |
| `Feedback Responses` | Source of monthly feedback/project update submissions. |
| `KPI Summary` | Calculated metrics for reports and dashboard. |
| `Dashboard` | Visual dashboard for chart snapshots and PDF export. |
| `Report Log` | Tracks reports sent, PDF links, and errors. |

## Design Rules

- Version 1 is email-only.
- No WhatsApp API in version 1.
- No individual 1:1 reminders in version 1.
- No participant names in stakeholder reports.
- Use anonymous mentee IDs in reports.
- Apps Script calculates metrics first.
- Gemini only writes narrative from structured, calculated metrics.
