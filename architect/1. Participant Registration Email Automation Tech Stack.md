# 1. Participant Registration Email Automation Tech Stack

## Purpose

Send registration emails to mentees after they apply, including a separate email for mentees who registered before the automation was installed.

## Tech Stack

| Layer | Tool | Why |
| --- | --- | --- |
| Source data | Google Sheets response tab | The mentee application data already lives in Google Sheets. |
| Automation runtime | Google Apps Script V8 | Native to Google Sheets and easy to trigger from form submissions. |
| Email sending | `MailApp.sendEmail` | Reliable built-in Apps Script email sender. |
| Sheet controls | Custom Google Sheets menu | Gives the team buttons for setup, test, preview, bulk send, selected-row send, and resend. |
| Trigger | Installable `onFormSubmit` trigger | Sends emails automatically for new registrations. |
| Duplicate protection | Status columns in the response sheet | Prevents accidental bulk duplicate emails. |
| Deployment | `clasp.cmd` | Pushes local files to Apps Script. |
| Version control | GitHub | Tracks script and documentation history. |
