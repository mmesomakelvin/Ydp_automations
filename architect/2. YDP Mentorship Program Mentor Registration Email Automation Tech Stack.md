# 2. YDP Mentorship Program Mentor Registration Email Automation Tech Stack

## Purpose

Send registration emails to mentors after they apply, including a separate email for mentors who registered before the automation was installed.

## Tech Stack

| Layer | Tool | Why |
| --- | --- | --- |
| Source data | Google Sheets response tab | The mentor application data already lives in Google Sheets. |
| Automation runtime | Google Apps Script V8 | Native to Google Sheets and supports form-submit triggers. |
| Email sending | `MailApp.sendEmail` | Built-in Apps Script email sender. |
| Sheet controls | Custom Google Sheets menu | Gives the team safe manual controls. |
| Trigger | Installable `onFormSubmit` trigger | Sends emails automatically for new mentor registrations. |
| Duplicate protection | Status columns in the response sheet | Prevents accidental duplicate emails. |
| Deployment | `clasp.cmd` from the `mentor/` folder | Keeps the mentor script separate from the mentee script. |
| Version control | GitHub | Tracks script and documentation history. |
